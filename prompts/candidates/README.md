# Prompt Candidates

Prompts promovidos de exploration, em avaliação para o registry de produção.

## Critérios para promover ao Registry

- Testado em ≥2 cenários
- Sem regressões
- Prompt estável (sem iterações pendentes)

## Como promover

1. Adicione entrada em `registry/project.yaml`
2. Defina `verified: true`
3. Remova ou arquive aqui
