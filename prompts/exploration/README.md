# Prompt Exploration

Rascunhos e experimentos de prompts. Não afeta produção.

## Como usar

1. Copie `_template.yaml` como `[seu-id].yaml`
2. Preencha id, name, prompt, hypothesis
3. Itere livremente
4. Quando estabilizar, promova para `candidates/` (ver WORKFLOW.md)

## Arquivos

- `_template.yaml` — template para novos explorations
- `*.yaml` — explorations ativos
